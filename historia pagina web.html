Historia del Internet y las Páginas Web

El nacimiento del Internet

El Internet, como lo conocemos hoy en día, comenzó a desarrollarse en la década de 1960 con la creación de ARPANET, un proyecto del Departamento de Defensa de los Estados Unidos. ARPANET fue la primera red que implementó el protocolo TCP/IP, que es la base de las comunicaciones en Internet.

Las primeras páginas web

Las páginas web nacieron en el año 1991 cuando Tim Berners-Lee, un científico británico, creó la primera página web en la CERN (Organización Europea para la Investigación Nuclear). Esta página web era muy simple y contenía solo texto y enlaces, sin ningún tipo de interactividad.

El Año de Nacimiento: 1991

El 6 de agosto de 1991, Tim Berners-Lee publicó la primera página web, que explicaba el proyecto World Wide Web y cómo utilizar un navegador web para acceder a documentos en línea.

La evolución de la interactividad

Las páginas web empezaron a volverse interactivas a mediados de la década de 1990 con la introducción de JavaScript, un lenguaje de programación que permitía a los desarrolladores crear páginas web dinámicas y responsivas.

Interactividad: JavaScript y CGI

Con la aparición de tecnologías como JavaScript en 1995 y el uso de CGI (Common Gateway Interface) para interactuar con servidores, las páginas web pudieron ofrecer una experiencia más dinámica y personalizada para los usuarios.

Versiones de una página web

1.0: La Web Estática

Las primeras versiones de las páginas web eran estáticas, lo que significa que el contenido era fijo y no cambiaba a menos que se actualizara manualmente por el desarrollador. Esto se conoce como Web 1.0.

2.0: La Web Dinámica

La Web 2.0 surgió a principios de los 2000 y se caracteriza por la interactividad, la colaboración y el contenido generado por los usuarios. Las redes sociales, los blogs y las wikis son ejemplos de Web 2.0.
